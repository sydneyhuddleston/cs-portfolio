How-To Make a Calculator

Using HTML, Javascript, and CSS

What can my Calculator do?

My calculator can add, subtract, multiply, divide, calculate exponents and percentages based on the two values entered and the chosen operation.

How can you make this calculator?

This calculator was created in three sections: HTML, then Javascript, then CSS. Below the instructions for each section is an image of my full code with comments that indicate which step was completed.

HTML

  1. Set up HTML document in proper HTML format.
  2. Link your CSS document to HTML document.
  3. Create a heading that will help the user identify your page. I used the <h1> and <center> tags to add the phrase "Sydney's Calculator".
  4. Insert the input text box. This can be done using the <input> tag. Within the tag, identify the type of input and create an ID for the input box. The input will be a number and your ID can be anything you want. You will use this ID in the Javascript Section.
  5. Insert a dropdown menu for your different operations using the <select> tag. Within this tag, create an ID as you did in the step above. Using the <option> tag, create an option for each operation.
  6. Identify option values in list form. Make sure you do not have more than one operation per option. This list determines the order the user will view the operations in, from top to bottom.
  7. Insert Second Input text box. Use the same method as the first input box, using the <input> tag. Like the first box, the value of the input will be numbers. Be sure to not make the ID of the second box identical to that of the first.
  8. Create the "Solve" button that will be clicked when the user has inputted the desired information. Using the same <input> tag, create a unique ID for the button that has not been used previously. The value of the button will be "solve".
  9. Use the <div> tag to create a div for your button. Create a unique ID for the div within the tag.
  10. Link your javascript to your HTML document using the script tag. The type is "javascript" and the source is the name of your Javascript Document.
  11. Be sure to proofread your code, making sure all tags are closed and there are not two different values with the same ID name.

Javascript

  1. Define all variables using the getElementById method. The format should read: var name = getElementById(HTML ID Name). Do this for all HTML buttons and text boxes that have an ID.
  2. Use the addEventListener method to activate the Solve button.
  3. Create a new variable for each text box. Because we do not know exactly what the user will input, it is a variable. using the parseInt
  4. Create a variable for the dropdown menu.
  5. Create if-statements for each of the potential dropdown menu options. The format for an if-statement is seen in the image on the left below. The condition determines which dropdown option will be used. The codeblock determines what to do if this option is selected. The format of the codeblock is: div id .innerhtml = value 1 operation value 2. Complete this process for all dropdown menu options.
  6. Your calculator should now be functioning.

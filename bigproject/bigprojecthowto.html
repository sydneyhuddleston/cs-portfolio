The Javascript Behind Making a Countdown Clock

What is a countdown clock?

A countdown clock is a timer that can count down by seconds to a desired date from the current moment in time.

Why did I make a countdown clock?

I made a countdown clock to count down to the day of my graduation because I am excited for the event.

Javascript Instructions

  1. Below is an image of my full code along with comments that indicate where each step was executed. It may be helpful to use it as a guide when reading these instructions if you would like a visual.
  2. Identify the desired countdown date by creating a variable to represent it. Your date should be formatted as such: first three letters of month name then day of month then comma then year then exact time in hours:minutes:seconds. The newDate() object will initiate this date. The .getTime() method will calculate the number of milliseconds at this moment in time.
  3. Identify when you want your countdown to refresh. I chose to have my countdown refresh every one second. I created a variable then used the setInterval method to set the countdown interval to be one second. Inside of the function will be the calculations that will determine the time differential between today's date and the desired countdown date.
  4. Making sure to put this code as the function of the countdown interval, define today's date using the new Date() object as we did in the first step. Then use the .getTime() method to calculate the number of milliseconds at this current moment in time.
  5. Then, create a variable that subtracts the number of milliseconds of the current moment in time from the number of milliseconds of the desired countdown date. It will look as if the new variable is equal to the variable that represents today's time being subtracted from the variable that represents the countdown day's time.
  6. Because the difference between the desired date and current date are currently in millisecond format, we must convert their values into days, hours, minutes, and seconds. I found these exact equations on a stackoverflow discussion page. We will create a variable for each of the different lengths of time: one for days, one for hours, etc. Using the Math.floor method, our variable will return an integer. We will divide our difference by their value during one day. For example, we will divide the difference of hours by 1000 multiplied by sixty and multiplied by sixty again. This is because there are 1000 milliseconds in one second, multiplied by sixty seconds in one minute, multiplied by sixty minuted in one hour, then the modulus of this new value to calculate the number of minutes for that specific hour.
  7. Repeat the above step for days, hours, minutes, and seconds.
  8. Now, we will use the document.getElementByID method to display the value of the above variables added together.
  9. You now have a functioning countdown timer. As an optional step, you can have your timer turn a different color when it has reached your desired date using an if statement. You should create a variable representing your countdown timer display and change the style of that variable to be your desired color when the time difference is below zero.
